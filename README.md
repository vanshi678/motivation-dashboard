# Motivation Dashboard

Motivation Dashboard is a simple React web application that displays motivational quotes to users.
Users can like quotes and view their liked quotes in a separate section.

## Live Demo

https://motivation-dashboard-roan.vercel.app/

## Features

* Display motivational quotes
* Like your favorite quotes
* View liked quotes
* Simple and clean user interface
* Built using React functional components and hooks

## Technologies Used

* React.js
* JavaScript
* HTML
* CSS
